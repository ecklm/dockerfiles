# Mirror ranking for pacman in docker

This docker image helps runs the official pacman mirror ranking program which
ranks mirrors according to download speed from them.

You can pass arguments in the run command, try `docker run -it
ecklm/pacman-rankmirrors -h` to see what your options are.

For info about the backgrounds, see the [ArchWiki](https://wiki.archlinux.org/index.php/Mirrors#List_by_speed)
